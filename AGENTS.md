# AGENTS.md

## Project overview
- This workspace contains a very small static website built from simple HTML files.
- The primary files are [hello.html](hello.html) and [welcome.html](welcome.html).
- There is no build step, package manager, or automated test suite in this repository.

## Working conventions
- Keep changes small, clear, and limited to the relevant HTML file.
- Preserve the existing structure and relative links unless a change explicitly requires updating them.
- Prefer semantic HTML and simple, readable markup over unnecessary abstractions.
- When editing, assume the editor may autosave changes; review the final file content before finishing to avoid leaving partial or broken edits.

## Verification
- After editing HTML, verify that the file remains valid and that any links or paths still resolve correctly.
- If content is changed significantly, preview the page in a browser to confirm the result looks correct.
